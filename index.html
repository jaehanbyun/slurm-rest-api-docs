Interactive API documentation for Slurm Workload Manager

About This API

This is an interactive documentation for the Slurm REST API (slurmrestd). The API provides programmatic access to Slurm workload manager functionality.

Authentication

  • API Key: Use X-SLURM-USER-NAME and X-SLURM-USER-TOKEN headers
  • Basic Auth: Standard HTTP Basic Authentication
